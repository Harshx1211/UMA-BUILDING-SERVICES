# UMA Building Services — Brand Assets

## Folder Contents

| File | Description |
|------|-------------|
| `uma_logo.svg` | Primary vector logo — scalable, use this for all digital/print use |
| `business_card.html` | Interactive business card preview — open in any browser |
| `assets/uma_logo.png` | Raster logo snapshot (backup) |
| `assets/uma_card_front.png` | Card front reference image |
| `assets/uma_card_back.png` | Card back reference image |

## Brand Colours

| Name | Hex | Use |
|------|-----|-----|
| Navy (Primary) | `#111D3F` | Backgrounds, headings |
| Navy Mid | `#1E2F62` | Card surfaces, UI panels |
| Flame Orange | `#D4561A` | Accents, CTAs, icon |
| Orange Light | `#E86A28` | Hover states, sub-accents |
| Silver | `#9DAABB` | Body text on dark |
| Dim | `#5A6880` | Labels, captions |

## Typography

- **Headings / Name:** Cormorant Garamond (serif, elegant)
- **Body / UI:** Inter (sans-serif, clean)
- **Logo mark text:** Arial Black / Inter Black (900 weight)

## Printing the Business Card

1. Open `business_card.html` in Chrome or Edge
2. Press `Ctrl + P` (or use the "Print / Export PDF" button)
3. Set: **No margins**, **Background graphics ON**, **Scale: 100%**
4. Save as PDF or send to printer
